Sobre o projeto

Este site foi feito para rodar no GitHub Pages e usar um backend separado no Render apenas para proteger a chave do Gemini.

Por que tem backend?

Porque chave de API não pode ficar no GitHub Pages. O site chama o Render, e o Render chama o Gemini usando a variável segura GEMINI_API_KEY.

As questões são inventadas?

Não. O site busca questões de uma API pública comunitária com questões do ENEM. A classificação por matéria, assunto e dificuldade é estimada para organizar o estudo.

Matérias usadas

Física, Química, Biologia, Matemática, Linguagens, Filosofia, Sociologia, História e Geografia.

Como publicar

Suba a pasta github-pages-site no repositório do GitHub Pages e a pasta render-backend em outro repositório conectado ao Render.
